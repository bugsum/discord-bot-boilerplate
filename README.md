## Usage

> Create a `config/config.json` file in the `src/` folder, paste the config data provided below & edit it as per your needs.

```json
{
	"name": "YourAppName",
	"environ": "prod",
	"developers": ["firstBotDevID", "secondBotDevID"],
	"dev": {
		"client": {
			"id": "ClientID",
			"guild": {
				"id": "GuildID",
				"invite": "https://discord.gg/GuildInvitation"
			}
		}
	},
	"prod": {
		"client": {
			"id": "ClientID",
			"guild": {
				"id": "GuildID",
				"invite": "https://discord.gg/GuildInvitation"
			}
	}
}
```
